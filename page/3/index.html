Flex 布局是什么

Flex 是 Flexble Box 的缩写，意为 “弹性布局”，用来为盒状提供最大的灵活性。

任何一个容器都可以指定为 Flex 布局

注意，设为 Flex 布局以后，子元素 的 loadt、cleat 和 vertical-align 将失去作用
